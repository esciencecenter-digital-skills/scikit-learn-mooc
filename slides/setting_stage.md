# The goal of this workshop

<img src="../figures/scikit-learn-logo.svg">

Building a machine learning pipeline using scikit-learn from start to end.

---
# What we will cover

- Exploring and preparing data
- Training and evaluating a machine learning model
- Understanding model performance and selecting the best model
- Tuning model hyperparameters

---
# What we will not cover

- Mathematical details (e.g., equations)
- Specific machine learning models
- Deep learning and neural networks
- Explainability
